# solar_home_lighting hardware files

This folder is meant to host files for the PCB and ESP32 devices as part of the overall Solar Home Lighting Project. 

## how to add

figure out how to get git on your local computer (i use git bash with VSCode for the app, there are youtube tutorials for this).

### for VSCode users:

open a project in VSC.

copy the repo and paste in terminal:
git clone https://github.com/popochom/Solar-Home-Lighting

make a folder with your name and/or subsystem.

#### (optional but good practice) add a README.md with a brief description on your files and what they do.

add all relevant files and documents, create sub-folders for organization if necessary. 

### once everything is ready, type in terminal:

  git add hardware_files/*
  
  git commit -m "added my files"
  
  git push origin main
  
#### Note: if its been a while since you cloned, make sure to pull before pushing:
  git pull origin main
